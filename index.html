About Ceawler

Ceawler is a powerful open-source tool designed to help you gather information effortlessly. Whether you're searching by username, email, phone number, or a custom query, Ceawler leverages APIs like DaData and Google Custom Search to deliver fast, reliable results.

Inspired by modern tools like GitHub Copilot, Ceawler combines a sleek interface with robust functionality, making it a must-have for developers, researchers, and enthusiasts.

Features

Username Search

Scan platforms like GitHub, Instagram, and Twitter for a given username.

Email Lookup

Check email presence across services like HaveIBeenPwned and Gravatar.

Phone Verification

Validate phone numbers and get detailed info using DaData API.

Browser Search

Search the web with custom queries via Google Custom Search.

Get Started

Ready to explore Ceawler? Clone the repository, set up your API keys, and start searching!

git clone https://github.com/ваш_ник/ceawler.git
cd ceawler
pip install -r requirements.txt

Create a config.py file with your API keys and run:

python main.py --phone "+79270000000"
